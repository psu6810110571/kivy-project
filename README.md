# 💣 Kivy Time-Attack Quiz Game

แอปพลิเคชันเกมตอบคำถามแบบจับเวลา พัฒนาด้วย **Python** และ **Kivy Framework** เป็นส่วนหนึ่งของโปรเจกต์รายวิชา

---

## 🎮 ภาพรวมของโปรเจกต์ (Project Overview)

ตัวแอปพลิเคชันคือเกมตอบคำถาม (Quiz Game) ที่ผู้เล่นจะต้องแข่งกับเวลาที่นับถอยหลังในรูปแบบของ **"ระเบิดเวลา"**  
หากตอบช้าหรือตอบผิด ระเบิดจะทำงานทันที

ตัวเกมเน้นความตื่นเต้น ท้าทาย และการคิดอย่างรวดเร็วภายใต้ความกดดันของเวลา

---

## ✨ ฟีเจอร์หลัก (Features)

- **ระบบหมวดหมู่คำถาม:**  
  มีให้เลือกเล่นอย่างน้อย 3 หมวดหมู่ เช่น  
  - ความรู้ทั่วไป  
  - สุขภาพ  
  - วิชาการ  

- **ระบบระเบิดเวลา (Bomb Timer):**  
  มี UI แสดงการนับถอยหลังของเวลา เพื่อเพิ่มความกดดันให้ผู้เล่น

- **ระบบตัวช่วย (Hint System):**  
  ผู้เล่นสามารถกดปุ่มเพื่อขอคำใบ้ในการตอบคำถาม

- **สรุปผลคะแนน:**  
  เมื่อจบเกม ระบบจะแสดงคะแนนรวมที่ผู้เล่นทำได้

---

## 🛠 ข้อมูลทางเทคนิค (Technical Details)

โปรเจกต์นี้พัฒนาตาม Requirement ของการสร้าง **Kivy Application** ดังนี้

- **Widgets:**  
  ใช้งาน Widgets รวมกันไม่น้อยกว่า **30 Widgets**  
  (เช่น Button, Label, Image, ProgressBar, ScreenManager)

- **Callbacks:**  
  มีการใช้งาน Callback function ไม่น้อยกว่า **10 Callbacks**  
  เช่น  
  - ฟังก์ชันตรวจคำตอบ  
  - เริ่มจับเวลา  
  - เปลี่ยนหน้าจอ  
  - ขอคำใบ้  

- **Version Control:**  
  จัดการโค้ดผ่าน **Git และ GitHub**  
  ทำงานร่วมกันเป็นทีมโดยใช้หลักการ **Commit Early และ Commit Often**
