# 💣 Kivy Time-Attack Quiz Game 

แอปพลิเคชันเกมตอบคำถามจับเวลาระทึกขวัญ พัฒนาด้วย Python และ Kivy Framework เป็นส่วนหนึ่งของโปรเจกต์รายวิชา

## 🎮 ภาพรวมของโปรเจกต์ (Project Overview)
ตัวแอปพลิเคชันคือเกมตอบคำถาม (Quiz Game) ที่ผู้เล่นจะต้องแข่งกับเวลาที่นับถอยหลังในรูปแบบของ "ระเบิดเวลา" หากตอบช้าหรือตอบผิด ระเบิดจะทำงาน! โดยตัวเกมเน้นความตื่นเต้นท้าทาย และมีฟีเจอร์ตัวช่วย (Hint) สุดพิเศษ

### ✨ ฟีเจอร์หลัก (Features)
* **ระบบหมวดหมู่คำถาม:** มีให้เลือกเล่นอย่างน้อย 3 หมวดหมู่ (เช่น ความรู้ทั่วไป, สุขภาพ, ฯลฯ)
* **ระบบระเบิดเวลา (Bomb Timer):** มี UI และเสียงแจ้งเตือนเมื่อเวลาใกล้หมด เพื่อเพิ่มความกดดันให้ผู้เล่น
* **ระบบตัวช่วย (Hint System):** ผู้เล่นสามารถกดขอคำใบ้ได้ โดยจะแสดงผลพร้อม UI รูปอาจารย์ธนาธิป 
* **สรุปผลคะแนน:** เมื่อจบเกมจะมีการสรุปคะแนนที่ทำได้

## 🛠 ข้อมูลทางเทคนิค (Technical Details)
โปรเจกต์นี้พัฒนาตาม Requirement ของการสร้าง Kivy Application ดังนี้:
* **Widgets:** ใช้งาน Widget รวมกันไม่น้อยกว่า 30 Widgets (เช่น Button, Label, Image, ProgressBar, ScreenManager)
* **Callbacks:** มีการใช้งาน Callback function ไม่น้อยกว่า 10 Callbacks (เช่น ฟังก์ชันตรวจคำตอบ, เริ่มจับเวลา, เปลี่ยนหน้าจอ, ขอคำใบ้)
* **Version Control:** จัดการโค้ดผ่าน Git และ GitHub ทำงานร่วมกันเป็นทีมโดยใช้หลักการ Commit Early & Commit Often (เป้าหมายรวม > 75 Commits)

## 🚀 วิธีการรันโปรแกรม (How to Run)
1. ตรวจสอบให้แน่ใจว่าติดตั้ง Python เรียบร้อยแล้ว
2. ทำการ Clone repository นี้ลงเครื่อง:
   `git clone https://github.com/psu6810110571/kivy-project.git`
3. ติดตั้งไลบรารี Kivy:
   `pip install kivy`
4. รันไฟล์หลักเพื่อเริ่มเกม:
   `python main.py`
