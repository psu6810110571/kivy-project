# 💣 BOMB QUIZ — Time-Attack Quiz Game

> เกมตอบคำถามจับเวลา สไตล์กู้ระเบิด | พัฒนาด้วย Python + Kivy

---

## 🎮 เกมนี้คืออะไร?

คุณคือ **เจ้าหน้าที่กู้ระเบิด** ที่ต้องตอบคำถามเพื่อตัดสายไฟให้ถูกต้องก่อนเวลาจะหมด  
ตอบช้า ตอบผิด หรือเวลาหมด — **ระเบิดทำงานทันที! 💥**

---

## ✨ ฟีเจอร์เด่น

- 🔌 **ตัดสายไฟโดยตรง** — แตะบนรูประเบิดได้เลย ไม่ใช่แค่กดปุ่ม
- 💥 **แอนิเมชันระเบิด** — สั่นเมื่อใกล้หมดเวลา, ระเบิดเมื่อแพ้
- 🔥 **Combo System** — ตอบถูกต่อเนื่อง คะแนนทวีคูณ
- ⚡ **Speed Run Bonus** — ตอบไวได้เปรียบ! ระบบคำนวณคะแนนพิเศษตามเปอร์เซ็นต์เวลาที่เหลือ
- 💡 **ระบบคำใบ้ (Hint)** — ยอมสละคะแนน 20% เพื่อแลกกับคำใบ้เอาตัวรอด
- ⏭️ **ระบบข้ามคำถาม (Skip)** — ยอมเสียหัวใจ 1 ดวงเพื่อข้ามข้อที่ทำไม่ได้ โดยรักษา Combo ไว้
- 🎲 **Lucky Dodge** — โอกาส 10% ปาฏิหาริย์รอดตาย! ระเบิดขัดข้องทำให้ไม่เสียชีวิตเมื่อตอบผิด
- 🏆 **Leaderboard + Achievements** — บันทึกสถิติและปลดล็อกรางวัล
- 👥 **โหมด 2 ผู้เล่น** — ผลัดกันเล่นบนเครื่องเดียว แข่งขันกันจุใจ 12 ข้อ
- 🕵️ **Secret Cheat Code** — รหัสลับซ่อนอยู่ (ลองใช้ชื่อสมาชิกในทีมเป็นชื่อผู้เล่นสิ! รับทันที 99 ชีวิตและคะแนน x2)

## 🆕 อัปเดตล่าสุด (v3.0 - Gameplay Mechanics Update)

- ✅ เพิ่มระบบคำใบ้ (Hint) พร้อมแอนิเมชันแจ้งเตือน
- ✅ เพิ่มหมวดคำถามใหม่ "ปริศนาเชาวน์ปัญญา" (Logic)
- ✅ แก้ระบบ 2 ผู้เล่น — สลับตาถูกต้องทุกกรณี (แยกชีวิตคนละ 3 ❤️)
- ✅ รองรับเต็มจอ + ปรับ Hitbox สายไฟแม่นขึ้น
- ✅ เพิ่มเพลง BGM หน้าเมนูหลัก
- ✅ Speed Run Bonus: ปรับสูตรคำนวณแต้มใหม่ ยิ่งกดตอบไว ยิ่งได้คะแนนโบนัสเพิ่ม (สูงสุด +150 แต้ม)
- ✅ Shuffle Choices: เพิ่มระบบสุ่มสลับตำแหน่งตัวเลือกคำตอบทุกครั้งที่เล่น ป้องกันการจำปุ่ม
- ✅ Skip Question: เพิ่มปุ่มข้ามคำถาม แลกกับ 1 ชีวิตเพื่อรักษา Combo สถิติ
- ✅ Perfect Clear Bonus: เพิ่มโบนัสพิเศษ +1,000 แต้ม หากเคลียร์ด่านได้โดยไม่เสียชีวิตเลย
- ✅ Lucky Dodge (10%): เพิ่มระบบ RNG สุ่มโอกาส 10% รอดตายจากระเบิดขัดข้องเมื่อตอบผิดหรือเวลาหมด
