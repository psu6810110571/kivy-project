# 💣 BOMB QUIZ: Time-Attack Quiz Game (v4.0)

แอปพลิเคชันเกมตอบคำถามจับเวลาระทึกขวัญ พัฒนาด้วยภาษา **Python** และ **Kivy Framework**  
เป็นส่วนหนึ่งของโปรเจกต์รายวิชาการพัฒนาแอปพลิเคชัน (Mobile Application Development)

---

## 🎮 ภาพรวมของโปรเจกต์ (Project Overview)

**BOMB QUIZ** คือเกมตอบคำถามแนวใหม่ที่ผสมผสานระหว่าง "เกมลับสมอง" และ "แอ็กชันแบบโต้ตอบ"  
ผู้เล่นต้องรับบทเป็น **เจ้าหน้าที่กู้ระเบิด** ที่ต้องตอบคำถามวิชาการและความรู้รอบตัว  
เพื่อตัดสายไฟให้ถูกต้องภายในเวลาที่กำหนด  

> หากตอบช้า ตอบผิด หรือเวลาหมด — **ระเบิดจะทำงานทันที!** 💥

ตัวเกมถูกออกแบบมาเพื่อสร้างความตื่นเต้น ท้าทาย และฝึกการตัดสินใจภายใต้แรงกดดัน

---

## ✨ ฟีเจอร์หลัก (Key Features)

### 🧩 ระบบเกมเพลย์ (Interactive Gameplay)
- **Wire-Cutting Interaction:** ระบบตอบคำถามสุดล้ำที่รองรับทั้งการกดปุ่มตัวเลือก และการ **"แตะที่ตำแหน่งปลายสายไฟ"** บนรูปภาพระเบิดโดยตรง
- **Dynamic Bomb UI:** วิดเจ็ตระเบิดที่มีแอนิเมชันสมจริง สั่นได้เมื่อเวลาใกล้หมด และมีเอฟเฟกต์การระเบิดเมื่อจบเกม
- **Combo System:** ยิ่งตอบถูกต่อเนื่อง คะแนนยิ่งทวีคูณ พร้อมเอฟเฟกต์การแสดงผลคอมโบที่สวยงาม

### 📚 หมวดหมู่และความยาก (Categories & Difficulty)
- **3 หมวดหมู่คำถาม:** ความรู้ทั่วไป, สุขภาพ, และ วิชาการ/IT
- **3 ระดับความยาก:**

| ระดับ | เวลาต่อข้อ | จำนวนสายไฟ | ตัวคูณคะแนน |
|-------|-----------|------------|------------|
| 🟢 Easy | 15 วินาที | 4 สายไฟ | x1.0 |
| 🟡 Medium | 10 วินาที | 5 สายไฟ | x1.5 |
| 🔴 Hard | 7 วินาที | 6 สายไฟ | x2.5 |

- **หลากหลายโหมด:** โหมดเดี่ยว, โหมด 2 ผู้เล่น (ผลัดกันเล่น), และ Sudden Death (ตอบผิดครั้งเดียวจบ)

### 📊 ระบบบันทึกผลและรางวัล (Systems)
- **Leaderboard:** ระบบจัดอันดับผู้เล่นที่ทำคะแนนสูงสุด บันทึกข้อมูลผ่านไฟล์ JSON
- **Achievement System:** ปลดล็อกเหรียญรางวัลเกียรติยศเมื่อบรรลุเป้าหมายพิเศษในเกม

## 🛠 ข้อมูลทางเทคนิค (Technical Details)

โปรเจกต์นี้พัฒนาตามข้อกำหนด (Requirements) ของการสร้าง Kivy Application อย่างครบถ้วน:

- **Widgets (>30 Widgets):** ใช้งาน Widgets หลากหลาย เช่น `ScreenManager`, `BoxLayout`, `GridLayout`, `ClockBombWidget`, `VignetteWidget`, `ProgressBar`, `Label`, และ `Button`

- **Callbacks (>10 Callbacks):** ระบบ Event-driven เช่น การตรวจคำตอบ, ระบบนับเวลาถอยหลังด้วย `Clock`, การเปลี่ยนหน้าจอพร้อม Transition และแอนิเมชันปุ่มกด

- **Graphics & Animation:** ใช้ Kivy `Canvas` วาดภาพกราฟิกแบบ Vector และใช้งาน Kivy `Animation` เพื่อสร้างประสบการณ์ผู้ใช้ที่ลื่นไหล

- **Version Control:** จัดการเวอร์ชันผ่าน Git และ GitHub ทำงานร่วมกันเป็นทีมด้วยแนวทาง Commit Early & Commit Often
