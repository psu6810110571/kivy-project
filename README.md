แอปพลิเคชันเกมตอบคำถามจับเวลาระทึกขวัญ พัฒนาด้วยภาษา Python และ Kivy Framework เป็นส่วนหนึ่งของโปรเจกต์รายวิชาการพัฒนาแอปพลิเคชัน

🎮 ภาพรวมของโปรเจกต์ (Project Overview)

BOMB QUIZ คือเกมตอบคำถามแนวใหม่ที่ผสมผสานระหว่าง เกมลับสมอง และ เกมแอ็กชันแบบโต้ตอบ
