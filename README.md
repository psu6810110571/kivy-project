# 💣 BOMB QUIZ — Time-Attack Quiz Game

> เกมตอบคำถามจับเวลา สไตล์กู้ระเบิด | พัฒนาด้วย Python + Kivy

---

## 🎮 เกมนี้คืออะไร?

คุณคือ **เจ้าหน้าที่กู้ระเบิด** ที่ต้องตอบคำถามเพื่อตัดสายไฟให้ถูกต้องก่อนเวลาจะหมด  
ตอบช้า ตอบผิด หรือเวลาหมด — **ระเบิดทำงานทันที! 💥**

---

## ✨ ฟีเจอร์เด่น

- 🔌 **ตัดสายไฟโดยตรง** — แตะบนรูประเบิดได้เลย ไม่ใช่แค่กดปุ่ม
- 💥 **แอนิเมชันระเบิด** — สั่นเมื่อใกล้หมดเวลา, ระเบิดเมื่อแพ้
- 🔥 **Combo System** — ตอบถูกต่อเนื่อง คะแนนทวีคูณ
- 🏆 **Leaderboard + Achievements** — บันทึกสถิติและปลดล็อกรางวัล
- 👥 **โหมด 2 ผู้เล่น** — ผลัดกันเล่นบนเครื่องเดียว 12 ข้อ

## 🎯 ระดับความยาก

| ระดับ | เวลา/ข้อ | สายไฟ | คะแนน x |
|---|---|---|---|
| 🟢 Easy | 15 วิ | 4 | ×1.0 |
| 🟡 Medium | 10 วิ | 5 | ×1.5 |
| 🔴 Hard | 7 วิ | 6 | ×2.5 |
| ☠️ Sudden Death | 8 วิ | 5 | ×3.0 |
| 📅 Daily Challenge | 10 วิ | 4 | ×2.0 |

> **Hard Mode:** สายไฟสับเปลี่ยนตำแหน่งทุก 2 วินาที!  
> **Sudden Death:** ตอบผิดแค่ครั้งเดียว = จบเกมทันที

## 🚀 วิธีรันโปรแกรม
```bash
# 1. Clone โปรเจกต์
git clone https://github.com/psu6810110571/kivy-project.git
cd kivy-project

# 2. ติดตั้ง Kivy
pip install kivy

# 3. เริ่มเกม!
python main.py
```

> ต้องการ **Python 3.10+**
